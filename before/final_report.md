# 基于深度学习的玉米粒计数：技术综述与前沿进展

## 一、调研背景与意义

玉米（*Zea mays* L.）是全球重要的粮食、饲料和工业原料作物。玉米穗粒数、千粒重等指标直接决定了最终的产量表现，精准的玉米粒计数不仅是**产量估测**的关键依据，也是**品种鉴定**、**品质分级**和**种子表型分析**的核心环节。

传统人工计数方式存在效率低、主观性强、易疲劳等问题，难以满足大规模育种和现代精准农业的高通量需求。近年来，深度学习在计算机视觉领域取得了突破性进展，基于卷积神经网络（CNN）和Transformer架构的目标检测与分割技术被广泛引入玉米粒计数任务，实现了从**静态图像计数**到**动态视频实时计数**的阶段跨越。

---

## 二、技术路线概览

### 2.1 传统图像处理方法（前深度学习时代）

早期玉米籽粒计数主要依赖传统计算机视觉技术：

| 方法 | 原理 | 局限性 |
|------|------|--------|
| **阈值分割** | 利用籽粒与背景的灰度/颜色差异进行二值化分割 | 易受光照不均影响 |
| **形态学处理** | 通过膨胀、腐蚀等操作分离粘连籽粒 | 对密集堆叠场景效果差 |
| **分水岭算法** | 基于距离变换的分水岭分割 | 易过分割或欠分割 |
| **连通域分析** | 计算连通域数量和几何参数（长轴、短轴等） | 无法处理严重重叠 |

汪珂等人利用阈值分割将亮色籽粒与黑色背景分离，配合形态学处理计算连通域的长轴与短轴，对粒长、粒宽测量的平均相对误差分别为 1.22%、3.34% 与 4.22%。

### 2.2 深度学习驱动的主流方法

当前基于深度学习的玉米粒计数方法主要分为以下几个技术流派：

| 流派 | 代表方法 | 核心思路 |
|------|----------|----------|
| **目标检测 + 计数** | YOLOv5/v8/v11、Faster R-CNN | 检测每颗籽粒边界框，统计数量 |
| **目标检测 + 多目标跟踪** | YOLOv8 + ByteTrack | 动态视频流中跟踪下落籽粒，避免重复计数 |
| **语义/实例分割** | Mask R-CNN、U-Net | 像素级分割后统计连通域 |
| **半监督学习** | DeepCorn | 利用少量标注数据 + 大量未标注数据 |
| **Transformer架构** | Swin Transformer | 层次化自注意力机制捕获全局-局部特征 |

## 三、核心模型与技术深度解析

### 3.1 YOLO 系列：实时检测的主力军

YOLO（You Only Look Once）系列已成为玉米粒检测与计数的**最主流框架**。

#### YOLOv8 + ByteTrack（动态下落计数系统）

**Maize Kernel Batch Counting System Based on YOLOv8-ByteTrack**（2025年发表于 PMC）是该方向的核心成果：

- **系统架构**：高速相机捕获籽粒下落动态视频 → YOLOv8 目标检测 → ByteTrack 多目标跟踪 → 实时计数输出
- **技术亮点**：
  - YOLOv8 的 C2f 模块采用多分支特征融合架构，对高速运动中的籽粒变形和旋转具有强判别力
  - ByteTrack 利用创新性的关联策略解决帧间 ID 切换问题
- **性能表现**：动态下落场景计数精度 **超过 99%**

#### LWCD-YOLO（轻量化玉米籽粒快速检测）

基于 YOLOv11n 的轻量级检测模型（Sun 等人，2025），专门针对籽粒实时检测需求设计：

- 在保持检测精度的同时大幅降低计算开销
- 适合在边缘端嵌入式设备部署

#### YOLO-light-pruned（幼苗计数与叶龄监测）

Jiang 等人（2026）提出剪枝后的轻量级 YOLO 模型，用于近地面和无人机 RGB 图像监测玉米幼苗：

- 多数据集验证：不同叶龄（rRMSE = 4.07%–7.27%）、不同分辨率（rRMSE = 3.06%–6.28%）、不同种植密度（rRMSE = 3.38%–10.82%）
- 集成幼苗计数与叶龄估计，支撑精准农业决策

### 3.2 DeepCorn：半监督深度学习方法

**DeepCorn**（Khaki, Pham, Han 等人，2021，发表于 *Knowledge-Based Systems*）是玉米粒计数领域的代表性半监督方法：

- **创新点**：结合少量标注图像与大量未标注图像进行联合训练，显著降低数据标注成本
- **任务**：高通量图像玉米粒计数与产量估计
- **技术基础**：采用卷积神经网络（CNN）提取特征，半监督机制提升泛化能力
- **同一团队相关工作**：Khaki 等人（2020）在 *Sensors* 上发表的卷积神经网络玉米粒检测与计数方法

### 3.3 Swin Transformer：视觉 Transformer 的引入

2025年发表于 *Scientific Reports* 的研究将 Swin Transformer 用于谷物计数：

- **层次化自注意力机制**：通过移位窗口捕获谷物的局部细节和全局上下文
- **可解释性**：结合 Grad-CAM 和 LIME 解释模型聚焦区域
- **性能**：准确率 **98%**，超越传统 CNN（ResNet-50）和 DINO 模型

### 3.4 实例分割方法

对于果穗上密集排列的玉米粒，实例分割可提供更精细的粒级分析：

- **Mask R-CNN** 改进版（Wang 等人，2024）：用于小麦籽粒表型参数检测，方法可迁移至玉米
- **混合分割模型**（PMC，2025）：针对粘连（sticky）玉米籽粒的精确分割与损伤检测

## 四、典型应用场景

### 场景一：果穗上籽粒计数

- **目标**：对收获后的玉米果穗拍照，统计每穗粒数、行粒数等
- **常用技术**：YOLO 检测 + 行列排列分析、点云三维重建
- **代表工作**：Hobbs 等人（2021，*Frontiers in Robotics and AI*）建立了一套完整的果穗籽粒计数数据集与方法

### 场景二：动态下落籽粒实时计数

- **目标**：在脱粒/输送过程中通过高速相机拍摄，实时统计下落籽粒数量
- **常用技术**：YOLOv8 + ByteTrack 多目标跟踪
- **优势**：在工厂流水线或考种环节实现 **非接触式自动化计数**

### 场景三：田间地表籽粒检测

- **目标**：玉米收获后，田间散落的籽粒检测与计数，评估收割损失
- **核心挑战**：深色阴影、秸秆遮挡、光照变化
- **代表工作**：刘晓航等人（2022，*智慧农业*）比较了多种深度学习算法在该场景下的表现

### 场景四：无人机/近地面遥感幼苗计数

- **目标**：玉米苗期植株数量及生长状态监测
- **常用技术**：YOLO 系列 + Kalman 滤波跟踪
- **性能**：YOLO-light-pruned 在无人机影像上幼苗计数 rRMSE 低至 3.06%

## 五、公开数据集与基准

当前玉米粒计数研究面临**公共数据集匮乏**的瓶颈。现有主要资源包括：

| 数据集名称 | 来源 | 说明 |
|-----------|------|------|
| **On-Ear Corn Kernel Dataset** | Hobbs 等（2021） | 聚焦果穗上籽粒的定位与计数，包含多种玉米品种 |
| **Corn 3 Classes Image Dataset** | Kaggle / Avuçlu 等（2025） | 包含 Chulpi Cancha、Indurata、Rugosa 三类玉米品种的 1050 张图像，用于品种分类 |
| **Maize Tassel / Wheat Ear 数据集** | 农业视觉竞赛 | 虽非专门玉米粒数据集，但对算法迁移验证有参考价值 |
| **私有数据集** | 各研究团队自建 | 现有研究大多依赖自采集私有数据，限制了算法横向比较 |

> ⚠️ **痛点**：业内尚未形成统一的、涵盖多品种与多模态的开源大型基准数据库。Hobbs 等人明确指出，以往研究的私有数据封闭性严重阻碍了算法间的公平对比。

## 六、关键挑战

### 6.1 密集遮挡与粘连问题
籽粒在果穗上紧密排列，相互遮挡严重；在地面散落时也存在堆叠粘连。传统方法在密集场景下漏检率高。

### 6.2 跨场景泛化能力差
现有模型在特定环境（固定光照、单一品种）下表现良好，但在**跨品种、跨场景**测试时性能明显衰减。深色阴影、秸秆遮挡、光照突变均会造成关键特征丢失。

### 6.3 边缘端部署受限
高精度模型（如大体积 CNN、ViT）计算开销大，而田间设备算力有限。如何在保持精度的同时实现模型轻量化，是工程落地的重要瓶颈。

### 6.4 动态场景中的目标跟踪挑战
在动态下落计数场景中，高速运动、目标 ID 频繁切换、下落速度变化等问题对多目标跟踪算法提出极高要求。

## 七、未来展望

### 7.1 视觉大模型与多模态融合
- 借鉴 **SAM（Segment Anything Model）**、**DINOv2** 等大规模预训练视觉模型，可大幅提升跨场景泛化能力
- 融合高光谱成像（HSI）与可见光成像，实现籽粒外部形态与内部品质（蛋白质、淀粉、油脂含量）的同步检测

### 7.2 多目标跟踪技术的持续优化
从 ByteTrack 到 BoT-SORT、StrongSORT 等更先进的跟踪范式，将进一步提升动态计数场景下的鲁棒性。

### 7.3 轻量化与边缘部署
- 知识蒸馏、网络剪枝、量化技术将推动高精度模型在嵌入式设备上的实时运行
- LWCD-YOLO、YOLO-light-pruned 等已展示了轻量化方向的有效性

### 7.4 标准化开源数据集建设
建立涵盖多品种、多光照条件、多视角的**玉米粒计数公开基准数据集**，将极大推动该领域的算法公平评估与创新。

### 7.5 可解释人工智能（XAI）
Grad-CAM、LIME 等可解释性技术的引入（如 Swin Transformer 研究中的实践），有助于增强模型的可信度和农业领域的接受度。

## 九、结论

基于深度学习的玉米粒计数已从实验室概念验证阶段迈入工程化部署阶段。**YOLO 系列模型**凭借其检测速度与精度的平衡优势，成为该领域最广泛采用的基准框架；**DeepCorn** 等半监督方法有效降低了标注成本；**Swin Transformer** 等视觉 Transformer 正在为高精度计数开辟新路径。未来，随着多模态融合、边缘端轻量化部署和大规模预训练模型的引入，玉米粒计数技术将在智慧育种、精准农业和粮食安全保障中发挥越来越重要的作用。
